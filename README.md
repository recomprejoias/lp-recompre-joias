# LP Recompre Joias

Landing page institucional da Recompre Joias desenvolvida com Next.js 16, React 19 e Emotion.

## Requisitos

- Node.js 20+
- npm 10+

## Desenvolvimento local

```bash
npm install
npm run dev
```

Aplicacao disponivel em `http://localhost:3000`.

## Build de producao

```bash
npm run build
```

Build validado com sucesso neste repositório.

## Deploy na Vercel

1. Importe este repositório na Vercel.
2. Mantenha o preset `Next.js` detectado automaticamente.
3. Use as configuracoes padrao:

```text
Install Command: npm install
Build Command: npm run build
Output Directory: .next
```

4. Publique o projeto.

Nao ha necessidade de `vercel.json` para o fluxo atual.
